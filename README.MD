## TS solution build

Install dependencies:

```bash
yarn
```

Development:

```bash
yarn start # host: http://localhost:8080/
```

Build:

```bash
yarn build
```

Build server:

```bash
yarn static # host: http://localhost:8000/
```

Run tests:

```bash
yarn test
```
